Word Attainer Help

What is Word Attainer?

Word Attainer is a program that lets you generate new words which you can use for whatever you'd like. The program can be tweaked to generate words that are betters matches for what you need (e.g. names of people or magic spells).

How do I generate words?

The generation process consists of four steps:

  1. Select lists of morphemes
  2. Adjust the generator settings
  3. Run the generator
  4. Save the words you want to keep

What is a morpheme?

A morpheme is one part of a word. Several morphemes make up a complete word.

For example, the morphemes Pe and ter could make up the name Peter. There are no specific rules as to what is a morpheme - it's up to you when you create morpheme lists.

Technically, a morpheme could be several words or entire sentences.

What is a generator?

A generator takes lists of morphemes as input and randomly stitches them together to create unique words (which we call results). The process can be adjusted to your liking by modifying the generator's settings.

How do I create morpheme lists?

Morpheme lists are just plain text files where each line in the file is a morpheme. It can be any filetype, as long as it is formatted in this manner.

If you're still unsure, save the list of morphemes as a .txt-file.

Filetypes like .doc (Microsoft Word) or .rtf (Rich Text File) are not plain text files, which means you might get weird results if you choose use them as morpheme lists.

What is a profile?

A profile, as a file, specifies the settings and morpheme lists a generator should use. It also contains the preferences for the program itself (e.g. how many results to generate or whether to automatically sort the results).

You can tweak settings and preferences to your needs and then export your profile and give it a name. The profile can then be used again or shared with other people.

Word Attainer profiles have the filetype .wap.

What does it mean to map morphemes to lists?

Normally when generating results, the generator picks a random morpheme list and then a random morpheme from that list. If you choose to map morphemes to the lists, the first morpheme will be picked from the first list, the second morpheme will be picked from the second list, the third from the third list, and so on...

If there is going to be more morphemes in a result than amount of lists, the first list will be selected again and the process is repeated from there.

How do I edit a result?

You can edit results individually before you save them. To do this, simply double-click the result you'd like to edit. When you have made the desired changes, hit Return.

What happens if I merge results?

If you generate some results and then click the Merge-button, you can select a file that contains results from a previous generation. The results from this file will then be imported and merged with your new results. Any duplicate results will be removed and you are free to save the new set of results.
