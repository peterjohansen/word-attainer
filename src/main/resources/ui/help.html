Word Attainer Help

How do I generate words?

The generation process consists of four steps:

  1. Select lists of morphemes
  2. Adjust the generator settings
  3. Run the generator
  4. Export the words you want to keep

What is a morpheme?

A morpheme is one part of a word. Several morphemes make up a complete word.
E.g. Pe and ter could make up the name "Peter".

What is a generator?

A generator takes lists of morphemes as input and randomly stitches them together to create a unique word. The process can be adjusted by changing the generator's settings.
